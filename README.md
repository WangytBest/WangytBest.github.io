博客！！！！！

1. 下载git仓库

切换到dev分支新建博客
master分支用于发布博客使用，切勿在master分支上进行开发

```shell
# git clone git@github.com:WangytBest/WangytBest.github.io.git
# cd WangytBest.github.io
# git checkout -t dev
```

2. npm install 

安装

```shell
# npm i 
```

3. themes目录为主题目录
