CSS实现元素水平垂直居中

在前端开发过程中，盒子居中是常常用到的。其中 ，居中又可以分为水平居中和垂直居中。块级元素的水平居中比较容易，直接设置元素的margin: 0 auto就可以实现，垂直居中相对来说是比较复杂一些的。

高度位置，使用绝对定位

.parentElem {
  position: relative;
}

.childElem {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}

父容器高度已知

且只有一个元素，只要使用相对定位即可

.parentElem {
  height: 200px;
}

.childElem {
  position: relative;
  top: 50%;
  transform: translateY(-50%);
}

水平居中方案

行内元素

  text-aligin: center;

块级元素，宽度固定

  margin: 0 auto;

块级元素，宽度不定

  margin: 0 auto;
